# Emoji Clicker: The Vibe Check

A fun incremental clicker game built with SvelteKit.

## Development

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`

## Testing

Run the test suite:

```bash
npm run test
```

Run tests in watch mode:

```bash
npm run test -- --watch
```

Run tests with UI:

```bash
npm run test:ui
```

## Build

```bash
npm run build
```

## Deployment

This project is configured for GitHub Pages. The GitHub Actions workflow will automatically build and deploy the site when you push to the `main` branch.

The site will be available at: `https://Yojimb0.github.io/emoji-clicker/`

## Features

- Click to earn vibes
- Buy buildings to generate passive income
- Unlock farming mechanics
- Rank up and evolve your emoji
- Save/load game progress

## Test Coverage

The project includes comprehensive tests for:
- Game logic (VPS calculation, click power, building costs)
- Save/load system
- Farming mechanics
- Integration tests for game flow

Run `npm run test` to see all tests pass.
